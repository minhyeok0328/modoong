현재 앱에서 API 연동된 부분을 전부 목업 데이터로 변경해야 합니다.
시연용으로 백엔드 없이 사용해야 하는 환경으로 전체 프로젝트를 리펙토링 해야한다.

```
// 모킹 데이터
const mocks = [
  {
    request: { query: GET_USER, variables: { id: '1' } },
    result: {
      data: {
        user: { id: '1', name: 'John Doe', email: 'john@example.com' }
      }
    }
  }
];
```

별도의 msw 설치는 번들 사이즈 증가 문제가 있으며 네트워크 레벨에서 동작하기 때문에 빌드 복잡하니 위와 같이 src/mocks/ 폴더를 참고하여 직접 변수에 목업 데이터를 생성해야 한다

public/ 경로에 있는 이미지 등을 적극 활용하여 목업이여도 동적으로 데이터 변하게 해야한다.
기존에 목업으로 만들어진 코드가 존재하니 이를 보고 참고해라.

각 데이터 타입 등은 기존 개발된
src/atoms
src/graphql

등등 graphql 연동된 interface나 쿼리 타입 등을 참고하여 한국어로 만들어야 한다. 
대부분 추상화가 이루어져 있기 때문에 코드 상세 내용을 철저히 분석하여 기존 코드의 수정을 최소화 해야 한다.

경우에 따라 기존에 구현되어 있는 utils/ 폴더에 있는 함수를 적극 활용해도 좋다
